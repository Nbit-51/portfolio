Selected projects

From fused GPU kernels to full-stack allocation systems.

⚡ flagship
Hydra Engine ↗ repo

High-performance LLM inference framework for TinyLlama-1.1B. Implements fused Triton JIT kernels for RMSNorm + residual ops, a GIL-free C++17 speculative decoding loop via LibTorch and PyBind11, AVX2 SIMD token verification, and NF4 quantization — all wired into a CUDA Graphs execution pipeline.

1.68× kernel speedup 68% VRAM BW reduction
C++17 CUDA Triton PyTorch PyBind11 AVX2 LibTorch
📄 research · targeting TMLR
KV Cache Compression Study ↗ repo

Empirical analysis comparing KV cache compression strategies — budget truncation, H2O token eviction, StreamingLLM, INT4/INT8 quantization, and sliding window attention — across OPT-1.3B, IBM Granite-1B, and Qwen2.5-1.5B. Documents architecture-specific failure modes including an H2O positional incoherence bug with RoPE models.

Python PyTorch Quantization LLMs ML Research OPT Qwen2.5
🔧 systems
navexa ARMv8-A Library ↗ repo

A collaborative NumPy-equivalent for ARMv8-A — high-performance C++ numerical computing primitives filling architecture gaps. Contributed the SME Matrix Acceleration module and Virtualization Helpers. Cross-compiled to AArch64 via CMake; all 109 tests passing.

LPG Distribution System ↗ repo

Priority-aware LPG cylinder allocation system replacing India's FCFS model. Implements pro-rata shortage splitting, dynamic anti-starvation ageing with priority score decay, and a crisis mode — aligned with PMUY and NFSA mandates. Full-stack with FastAPI backend, MySQL, and QR-based consumer tracking.
